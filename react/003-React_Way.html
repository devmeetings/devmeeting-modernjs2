Pierwszy komponent w React

Obecnie aplikacja jest całkiem czytelna, ale co dwie sekundy zastępujemy wszystkie elementy nowymi, pomimo że część z nich w ogóle się nie zmienia!

Dodatkowo powoduje to problem z zarządzaniem zaznaczeniem, focusem czy wartością wpisaną w input.

Aby rozwiązać ten problem musielibyśmy uaktualniać tylko tę część DOM, która się zmienia, ale wtedy kod straciłby na czytelności.


React proponuje inne rozwiązanie.

Musimy zainstalować odpowiednie zależności:

  $ npm i react react-dom -S
